Forsaken Realms AI — Model Configuration
==========================================

Current model:  qwen3:8b
Engine:         Ollama  (http://localhost:11434)
Embedding model: all-MiniLM-L6-v2  (sentence-transformers)

To switch models:
  1. Pull the new model:   ollama pull <modelname>
  2. Open config.json in the AI Assistant root folder
  3. Change "ollama_model" to the new model name
  4. Restart the app

Tested models for this hardware (FX-6300 + GTX 1050 Ti 4GB):
  qwen3:8b       — best quality, think mode off recommended
  llama3.2:3b    — faster, good for quick Q&A
  phi3.5:3.8b    — fast, strong at structured answers
